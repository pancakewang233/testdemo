# Vue2 H5 明细合计计算应用

一个移动端H5应用，包含三行明细输入框和一个合计输入框。

## 功能特性

- 三个明细输入框，支持数字输入
- 明细输入自动统计到合计输入框
- 修改合计输入框时，自动清空所有明细输入框
- 移动端H5适配，响应式设计

## 安装依赖

```bash
npm install
```

## 运行项目

```bash
npm run dev
```

项目将在 http://localhost:8080 启动

## 构建生产版本

```bash
npm run build
```

## 技术栈

- Vue 2.6.14
- Webpack 5
- Babel
